คลังสื่อการเรียนรู้สำหรับห้อง ป.4/2 ปีการศึกษา 2569 รวมสื่อจริง ใบงาน บทเรียน และกิจกรรม ให้ค้นหาและเปิดใช้งานได้ง่ายทั้งในห้องเรียนและที่บ้าน

เปิด Learning Hub อ่านรีวิวระบบ

รีวิวระบบ

จุดเด่นของ Learning Hub

01

ค้นหาสื่อได้เร็ว

หน้าแรกมีช่องค้นหาสำหรับสื่อ ใบงาน หรือบทเรียน ช่วยให้นักเรียนและครูเข้าถึงไฟล์ที่ต้องใช้ได้โดยไม่ต้องไล่เปิดหลายโฟลเดอร์

02

จัดหมวดตามรายวิชา

รวมสื่อจริงของ ป.4/2 เช่น STEM คณิตสืบเสาะ วิทยาการคอมพิวเตอร์ และการงานอาชีพ ทำให้การใช้งานเป็นระบบ

03

เหมาะกับผู้เรียน

หน้าเว็บใช้ภาพ สี และปุ่มที่เป็นมิตรกับนักเรียนระดับประถม เปิดผ่านมือถือได้ง่ายและลดความซับซ้อนในการเข้าสื่อ

04

มีหลังบ้านครู

ออกแบบให้ครูจัดการรายการสื่อและอัปเดตเนื้อหาได้ต่อเนื่อง ใช้เป็นคลังหลักฐานสื่อการสอนและนวัตกรรมดิจิทัลได้ในระยะยาว

ภาพรวมการใช้งาน

Learning Hub ทำหน้าที่เป็นหน้าเชื่อมกลางระหว่างผู้เรียนกับสื่อการเรียนรู้ของครู โดยเน้นการค้นหา การจัดหมวด และการเลือกสื่อแนะนำ เหมาะสำหรับใช้ประกอบการสอนในห้องเรียน ส่งต่อให้นักเรียนทบทวน และใช้เป็นหลักฐานการพัฒนาสื่อดิจิทัลของครู ICT

กระบวนการใช้งาน

  1. ครูเพิ่มหรือจัดหมวดสื่อ ใบงาน และลิงก์บทเรียนในระบบ
  2. นักเรียนเปิด Learning Hub แล้วค้นหาหรือเลือกสื่อแนะนำประจำสัปดาห์
  3. เปิดไฟล์ประกอบกิจกรรม เช่น วิดีโอ เอกสาร ใบงาน รูปภาพ หรือสไลด์
  4. ครูปรับปรุงรายการสื่อตามหน่วยการเรียนและใช้เป็นหลักฐานงานสอน

Evaluation

สรุปคุณค่าต่อการจัดการเรียนรู้

ต่อครู

ลดเวลาค้นสื่อก่อนสอน จัดระเบียบแหล่งเรียนรู้ให้เป็นระบบ และใช้แสดงผลงานดิจิทัลใน e-Portfolio ได้ชัดเจน

ต่อนักเรียน

เปิดสื่อได้ง่ายจากหน้าเดียว เห็นสื่อแนะนำที่เหมาะกับบทเรียน และสามารถทบทวนย้อนหลังผ่านมือถือหรือคอมพิวเตอร์

ต่อหลักฐานวิชาชีพ

สะท้อนการใช้ ICT เพื่อจัดการเรียนรู้ การพัฒนาสื่อ และการออกแบบระบบสนับสนุนผู้เรียนที่นำไปใช้จริง
